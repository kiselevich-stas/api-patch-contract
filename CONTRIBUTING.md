# Contributing

Thanks for your interest in contributing.

## Local setup

```bash
npm install
npm run check
```

## Development

```bash
npm run dev
```

## Tests

```bash
npm run test
npm run test:types
```

## Pull requests

Before opening a PR, please run:

```bash
npm run check
```

## Good first issues

- Add more framework examples
- Improve documentation
- Add edge-case tests for arrays
- Add more type-level tests
- Add benchmarks
